This folder only contains the relevant ray tracing shader code:
* BillboardRayTracer.rt.slang - Contains all shader kernels
* Helper.slang - A collection of helper functions to fetch billboard data
* RayDifferentials.slang - Ray Differential code
* Shadow.slang - This file includes the function to compute ray traced shadows with ray queries

The project was build with the NVIDIA falcor framework (https://github.com/NVIDIAGameWorks/Falcor).
The full project and additional explanation is available here: https://github.com/kopaka1822/Falcor/tree/billboards
